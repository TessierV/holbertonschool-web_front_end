## Json Server 

### 1. Install nvm (if not installed)
```curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash```

### 2. Activate nvm (make sure to close and reopen the terminal after this step)
```source ~/.nvm/nvm.sh```

### 3. Install Node.js version 18
```nvm install 18```

### 4. Use Node.js version 18
```nvm use 18```

### 5. Install JSON Server locally in your project
```npm install json-server```

### 6.  Start JSON Server using npx
```npx json-server --watch db.json```

### 7. For test exercises 10 to 12
```move you file into a public directory```
